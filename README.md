# Password Generator

This projects function is to generate a random password with selectable Elements.

## Sections
* Alert and Confirm Windows
* Text Area With Titles and Generate button.
## Selectable Elements
* Password Length (8-128)
* Include Uppercase
* Include Lowercase
* Include Numbers
* Include Symbols
