# Password Generator

This projects function is to generate a random password with selectable Elements.

## Sections
* Centered Password Form

Form where you can select your password elements.
## Elements
* Password Length (8-128)
* Include Uppercase
* Include Lowercase
* Include Numbers
* Include Symbols
